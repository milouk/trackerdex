# Screenshots

The README references three images expected here:

- `dark.png` — main dex grid in dark theme (default)
- `light.png` — same view, light theme
- `sprite-overview.png` — entry detail page (full-size sprite + stats)

Suggested capture: 1440×900, browser at the public host.

To take them by hand:

1. `npm run dev` against a real Pi-hole with a populated catch state
   (the more legendaries caught, the better the screenshot).
2. Use your OS's native screen capture or a browser tool. Save as PNG
   with the names above.

Once you've taken them, drop them in this directory and they'll appear
in the README automatically.
